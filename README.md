# 贪吃蛇 Snake

欢迎游玩【贪吃蛇 Snake】！

## 游戏简介

贪吃蛇是一款简单有趣的休闲小游戏。
玩家通过控制蛇不断吃掉食物让它变长，但要小心不能碰到墙壁或自己。

**此项目为 Web 实现版，采用现代简约风格配色与音效，兼容桌面与移动端体验。**

---

## 玩法说明

- 键盘方向键 `↑` `↓` `←` `→` 控制蛇的移动方向。
- 也可以用 `W/A/S/D` 键进行控制。
- 在移动端支持滑动手势操控：用手指在画布上按“滑动”即可转向。
- 蛇吃到红色食物会变长并得分。
- 分数大于 50 时游戏自动进入“超级模式”：
  - 蛇移动速度加快，棋盘背景变为深红色，增加紧张感。
- 撞到墙壁或自己则游戏失败。

---

## 如何运行

1. 使用现代浏览器（推荐 Chrome/Edge）直接双击/右键打开 `index.html` 即可开始游戏。
2. 如果需要背景音乐，请确保 `bgm.mp3` 音频文件与本项目在同一目录下。

---

## 文件结构

- `index.html` — 主页结构
- `style.css` — 游戏美术样式表
- `script.js` — 游戏JavaScript逻辑
- `bgm.mp3` — 背景音乐 (需自备放置)
